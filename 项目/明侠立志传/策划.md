# 简介

**明朝洪武年间，江湖与庙堂的边界从未如此模糊。**

你从一场灭门惨案中侥幸逃生，手中仅剩半本家传武学。是隐姓埋名、苦练神功，最终手刃仇敌？还是投入锦衣卫麾下，以权谋之道搅动风云？亦或开宗立派、悬壶济世，成为一代武林泰斗？

**《明侠立志传》** 是一款融合了开放江湖、多线剧情与高自由度角色养成的武侠模拟游戏。你将扮演一位出身平凡的少年/少女，在真实的明代社会体系中，**自由选择你的生存方式**：

* **学武**：投名师、偷秘笈、自创招式，从街头拳脚到绝世内功，数百种武学任你搭配。
* **入仕**：考取武举人、充当镖师、加入锦衣卫或东厂，借助朝廷之力完成复仇或野心。
* **经商**：开设武馆、贩卖丹药、经营盐铁生意，用财富买通人情或打造神兵利器。
* **游历**：结识一百余位江湖奇人，触发随机事件，探索大江南北的名山大川与秘境遗迹。

游戏采用**高自由度的每月日程制**，你可以随时切换身份——这个月在锦衣卫办案，下个月回山门教徒弟。你的每一次选择，都会影响大明武林的格局与你的最终结局。

是成为守护苍生的侠之大者，还是权倾朝野的黑暗枭雄？江湖路远，一切由你执笔。

> **特色一览：**
>
> * 十余种身份路线（镖师、捕快、大夫、铁匠、乞丐、刺客……）
> * 动态武林系统（NPC会自行成长、结仇、传功、陨落）
> * 多线程结局（达官显贵、归隐山林、开宗立派、以身殉道等20余种）
> * 明代真实风物（江南水乡、塞北荒漠、西南苗疆、京城皇城）

**残卷已在手，明月照我衣。你的江湖，从这里开始。**
