# 奇侠日记 策划文档

## 一、项目目标

* 减少 UI、特效、音效等美术成本，快速熟悉 Unity 2D 游戏开发流程
* 验证项目的发布、运营等能力
* 快速验证武功、装备、周目、门派、称号、天赋、钓鱼、AI 等玩法设计，积累初步开发经验

## 二、平台

安卓、iOS 等移动平台[](https://)

## 三、核心玩法概述

《奇侠日记》是一款轻量武侠题材的 Roguelike 放置游戏，核心玩法参考《冒险日记》。

**核心交互**：玩家每次操作仅需从三个选项中选择一个，选择后推进游戏进程并产生相应的结果反馈。

**核心循环**：

> 选择历练地点 → 触发事件/遭遇战斗 → 获取奖励（装备/秘籍/残章）→ 强化角色属性与武功 → 挑战更高难度内容 → 推动周目进程

## 四、日志系统

日志是游戏的主要信息展示窗口。所有事件反馈均通过日志滚动呈现。

### 日志类型与展示内容


| 类型     | 内容说明                                           | 示例                                                                       |
| -------- | -------------------------------------------------- | -------------------------------------------------------------------------- |
| **战斗** | 回合制战斗信息，包含武功名称、伤害数值、施加状态等 | "风云变色间，你施展了降龙十八掌，敌人失去12点生命，内脏翻滚，新增内伤状态" |
| **掉落** | 击败敌人后的物品获取信息                           | "你获得屠龙刀一把"（物品名称根据品阶显示对应颜色）                         |
| **任务** | 历练过程中触发的任务相关消息                       | —                                                                         |
| **系统** | 提示性信息，如传闻、引导等                         | "传闻在卧佛寺埋藏着连城诀的宝藏"                                           |

### 出招文本配置

武功出招描述文本挂载在武功数据上，每次出招时随机选取一条展示。

## 五、武学系统

### 5.1 武功类型

武功分为三类：


| 类型     | 说明                             | 装备数量 | 学习上限 | 备注                 |
| -------- | -------------------------------- | -------- | -------- | -------------------- |
| **外功** | 主要攻击手段，战斗中自动随机出招 | 3个      | 10个     | 特定事件可增加装备槽 |
| **内功** | 被动生效，决定外功伤害加成       | 1个      | 5个      | 特定事件可增加装备槽 |
| **轻功** | 影响行动力累积速度               | 1个      | 5个      | 特定事件可增加装备槽 |

### 5.2 武功属性


| 属性         | 说明                                                                                                                             |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------- |
| **威力**     | 决定武功基础伤害，上限为16，可通过 Buff 临时提升                                                                                 |
| **阴阳适性** | 每个武功分为阴/阳/中性之一，以0\~255数值表示（0=纯阴，255=纯阳，128=中性）。内功与外功阴阳相适则伤害加成，相悖则伤害减益         |
| **等级**     | 每个武功有独立等级，从"初涉"到"入圣"共七阶，每阶提供一个词条加成。高等级需通过特定任务或"武学残章"解锁，解锁前词条不可见、不生效 |
| **招式**     | 武功出招时可随机触发隐藏招式（如"亢龙有悔""行云密雨"），隐藏招式威力更强且附带特殊 Buff 效果                                     |
| **出招描述** | 每次使用武功时随机展示一条武学描述文本，用于增强沉浸感                                                                           |

### 5.3 武功等级表


| 等级 | 名称 | 说明               |
| ---- | ---- | ------------------ |
| 1    | 初涉 | 初窥门径，仅得皮毛 |
| 2    | 入流 | 登堂入室，已具章法 |
| 3    | 精进 | 日臻纯熟，渐入佳境 |
| 4    | 登峰 | 炉火纯青，罕逢敌手 |
| 5    | 造极 | 出神入化，一代宗师 |
| 6    | 超凡 | 超凡脱俗，近乎通玄 |
| 7    | 入圣 | 入圣超凡，神话传说 |

### 5.4 武功升级机制

* 武功初始获得时为"初涉"等级
* 每次战斗中使用该武功可累积熟练度
* 熟练度达标后，消耗"武学残章"可突破至下一等级
* 超凡、入圣等高等级除残章外，还需完成特定"悟道事件"方可解锁
* 被锁住的等级在 UI 中显示为 🔒 图标，悬停/点击可查看解锁条件

### 5.5 隐藏招式机制

* 触发概率：武功自带基础触发概率（建议15%\~25%），特定天赋可提升概率
* 触发时：本次攻击威力提升，且附加特殊 Buff（如内伤、眩晕、破甲等）
* 隐藏招式不额外消耗资源，视觉效果上应有明显区分

## 六、装备系统

装备分为武器、衣物、鞋子三类，每类提供不同的基础属性，并拥有独立的等级词条系统。

### 6.1 装备品阶与等级


| 颜色 | 名称 | 说明                   |
| ---- | ---- | ---------------------- |
| 白色 | 凡铁 | 粗制滥造，江湖寻常之物 |
| 绿色 | 青锋 | 锋芒初露，尚算趁手     |
| 蓝色 | 碧澜 | 碧波凝锋，颇为难得     |
| 紫色 | 紫霜 | 紫气凝霜，江湖罕见     |
| 紫色 | 金璃 | 金光琉璃，价值连城     |
| 粉色 | 绯烟 | 绯色如烟，缥缈若仙     |
| 红色 | 赤焰 | 赤焰灼天，神器之属     |

### 6.2 装备分类详情

**武器**：

* 基础属性：名称、描述、攻击力、暴击率
* 类型分为：剑、刀、奇门、棍、拳。使用与武器类型匹配的外功时有加成，不匹配时则大幅降低伤害

**衣物**：

* 基础属性：名称、描述、防御力
* 额外提供等级词条加成

**鞋子**：

* 基础属性：名称、描述、闪避率、行动力
* 行动力影响战斗中出招速度（累积至100时出招）

### 6.3 装备等级解锁机制

高品阶装备的词条默认被锁住，需消耗特定材料（如"玄铁""天蚕丝"）或完成特定挑战逐级解锁。锁住期间该词条不生效，但UI上会提示解锁条件。

## 七、天赋系统

天赋为角色提供特殊的词条效果，可增强特定武功、属性，或带来减益效果。

### 7.1 天赋品阶表


| 颜色 | 名称 | 说明                           |
| ---- | ---- | ------------------------------ |
| 白色 | 凡庸 | 平平无奇，江湖中人的普遍根骨   |
| 绿色 | 可造 | 尚可雕琢，具备习武的底子       |
| 蓝色 | 通明 | 心窍通达，悟性已超常人         |
| 紫色 | 卓绝 | 出类拔萃，百人之中难有一例     |
| 黄色 | 天纵 | 上天眷顾，生来便是练武奇才     |
| 粉色 | 惊世 | 惊才绝艳，足以震动武林         |
| 红色 | 造化 | 夺天地造化，近乎传说级别的天赋 |

### 7.2 资质限制机制

* 角色拥有"资质值"上限（初始建议为5\~10点）
* 每个天赋占用一定资质值（白色占1点，红色占7点，以此类推）
* 新增天赋时，若总占用超过资质值上限，则无法学习
* 资质值上限可通过特定事件或物品提升
* 允许"遗忘天赋"以腾出空间（有代价，如损耗少量属性）

## 八、人物属性

### 8.1 基础属性


| 属性 | 上限 | 说明                                       |
| ---- | ---- | ------------------------------------------ |
| 臂力 | 255  | 主要影响外功伤害（拳掌/刀法/剑法等）       |
| 身法 | 255  | 影响行动力累积速度，决定出手顺序           |
| 悟性 | 255  | 影响武功秘籍的学习速度                     |
| 根骨 | 255  | 影响内功修炼效果、内力上限及部分内功伤害   |
| 福缘 | 255  | 影响奇遇概率、稀有掉落率                   |
| 定力 | 255  | 影响对负面状态（内伤/眩晕/中毒）的抵抗能力 |

### 8.2 战斗属性

* **生命值**：归零则无法战斗。死亡时展示对应的遗言文本
* **内力值**：施展武功需消耗内力，不足时无法出招

### 8.3 阴阳属性

角色自身拥有阴阳值（0\~255），与武功/装备的阴阳适性共同影响伤害计算。

### 8.4 伤害计算流程

> ① 基础威力 × ② 阴阳适性系数 × ③ 内功加成系数 × ④ 等级词条系数 × ⑤ 武器类型匹配系数 = 最终伤害

## 九、敌人系统

敌人继承角色的全部属性系统，并额外配置以下内容：

* **掉落物品**：击败后可获得装备、秘籍、武学残章、材料等
* **遗言文本**：不同敌人/不同死亡方式配置不同遗言

### 9.1 战斗状态（Buff/Debuff）

负面状态分为5级，等级越高效果越强：


| 状态 | 效果说明                   |
| ---- | -------------------------- |
| 内伤 | 每回合损失生命，降低攻击力 |
| 眩晕 | 跳过行动回合，无法闪避     |
| 中毒 | 每回合损失生命，可叠加     |

## 十一、特殊事件


| 事件名称   | 触发条件                    | 结果说明                                                                                               |
| ---------- | --------------------------- | ------------------------------------------------------------------------------------------------------ |
| 跳崖事件   | 轻功 ≥ 60 + 当前地图为山地 | 可获得特殊武功或稀有物品                                                                               |
| 英雄救美   | 随机触发                    | 可能是真实救美，也可能是仙人跳（损失财物）                                                             |
| 偷窃少林   | 特定区域可触发              | 可获得特殊物品，也可能遭遇扫地僧（高难度战斗）。装备夜行衣可提高成功率                                 |
| 钓鱼       | 特定地图/消耗体力           | 钓到特定鱼类可永久提升属性                                                                             |
| 老乞丐事件 | 多次触发（共四阶段）        | 第一次：花费巨资获白板秘籍 → 第二次：获蓝色秘籍 → 第三次：获黄色秘籍 → 第四次：击败老乞丐获红色秘籍 |

每个特殊事件应明确标注是**一次性**还是**可重复触发**。

## 十二、周目系统

### 12.1 周目流程

游戏主线分为四个阶段：

> 门派学习 → 武林大会 → 除魔大会 → 华山论剑

华山论剑完成后，玩家可选择：

* **结束游戏**（结算评分）
* **进入下一周目**（保留继承内容，重置其他进度）

### 12.2 周目继承规则

* ✅ 保留：背包内所有物品
* ✅ 保留：武学已解锁的等级（但武功本身可能需要重新获取？建议明确）
* ❌ 重置：角色属性、装备穿戴状态、剧情进度、门派关系等

---

## 十三、门派系统

* 玩家可通过初始选择或事件触发加入门派
* 不同门派擅长的武功类型不同，提供独特的可学武功和门派任务
* 可设定"叛门"机制（退出当前门派加入新门派），伴随一定惩罚（如属性衰减、声望下降）

## 战斗系统

完整流程

> ① 根据轻功+身法计算初始行动力 → ② 行动力累积，谁先到100谁先出手 → ③ 从三个已装备外功中随机选一个 → ④ 判定命中 → ⑤ 计算伤害（阴阳适性+内功加成+等级词条）→ ⑥ 判定是否触发隐藏招式 → ⑦ 应用 Buff/Debuff → ⑧ 回合结束，下一方行动 → ⑨ 一方生命归零则战斗结束

---

## 十四、开发优先级建议

基于"快速验证"的目标，建议分阶段推进：

**第一阶段（最小可行版本）**：

* 三选一交互 + 日志展示
* 外功随机出招（简化版，无阴阳适性）
* 基础敌人 + 掉落

**第二阶段**：

* 武功等级 + 残章解锁
* 装备系统 + 等级词条
* 阴阳适性计算

**第三阶段**：

* 天赋系统 + 资质限制
* 内功/轻功装备
* 特殊事件

**第四阶段**：

* 周目系统
* 门派系统
* 钓鱼等补充玩法
